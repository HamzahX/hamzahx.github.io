Projects

Some software development projects I've worked on within the past year:

FootballSlices

Soccer stats scraper and visualization tool

I developed a series of scripts to automate the process of scraping data (with permission) from a popular soccer stats website. I also created a web application with a visualization tool to display the stats in my database.

Developed using: MERN Stack (MongoDB + Express + React.js + Node.js), Puppeteer, REST API


QuickSplash

A real-time multi-player party game (group project)

Drawing inspiration from JackBox Game's popular game Quiplash, we developed a similar game named QuickSplash. QuickSplash is a party game that revolves around giving the funniest answers you can to a collection of pre-defined prompts.

Developed using: MERN Stack (MongoDB + Express + React.js + Node.js), Socket.io, Bootstrap
